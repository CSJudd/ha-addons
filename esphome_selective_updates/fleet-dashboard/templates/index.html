Required Substitutions

For the fleet manager to work properly, all device YAMLs should include these substitutions:

Required:

  • device_type - Device hardware type (e.g., "Sonoff S31", "Athom Switch")
  • esphome_version - ESPHome version pinning ("*" for latest, or "2026.4.3" for specific)
  • area - Logical location/room name for grouping

Recommended (Optional):

  • physical_location - Exact physical install location (e.g., "Behind fridge, bottom left outlet") for maintenance
